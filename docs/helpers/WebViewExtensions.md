# HyperlinkExtensions

The **WebViewExtensions** allows attaching HTML content to WebView.

## Example

```xaml
	// Attach HTML content directly to WebView.
	<WebView xaml:WebViewExtensions.HTML="{Binding HTMLData}" />
```
